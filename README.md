# Chartjs

This gem packages the chart.js assets (JavaScripts) for the Rails 4+ 

## Usage

In your Gemfile, add:

```ruby
gem 'chartjs'
```

Or install:

```ruby
gem install chartjs
```

## Require Everything

To require all ChartJs modules, add the following to your application.js:

```javascript
//= require chart
```
